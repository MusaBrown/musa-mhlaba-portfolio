Research: Feature Engineering for Fraud Detection

Honours Research

Feature Engineering for Fraud Detection in Financial Transactions

Research Overview

This research analyses how feature engineering techniques influence the performance of machine learning models for financial fraud detection, comparing PCA and Mutual Information on Random Forest and Isolation Forest models.

Key Findings

Mutual Information consistently yielded superior results across both models, demonstrating that preserving the most relevant original features is more effective than creating new synthetic components through PCA.

Recommendation

For developing a real-world, production-ready fraud detection system, employing a supervised model (Random Forest) in combination with Mutual Information feature selection is essential.

Dataset

Kaggle Credit Card Fraud Detection (284,807 transactions, 0.172% fraud rate)

Methods

PCA vs Mutual Information | Random Forest vs Isolation Forest

Metrics

Accuracy, Precision, Recall, F1-Score, ROC-AUC
